## About The Repository
This repository contains a vast collection of end-to-end Machine Learning pipelines and AI models built for predictive analytics, natural language processing, and computer vision.

### Key Domains
1. **Supervised Learning:** Regression, Classification (XGBoost, Random Forests, SVMs).
2. **Unsupervised Learning:** Clustering (K-Means, DBSCAN), Dimensionality Reduction (PCA, t-SNE).
3. **Deep Learning:** Neural Networks, CNNs for Image Recognition, LSTMs and Transformers for NLP.
4. **Model Deployment:** Exporting models via ONNX, serving predictions via FastAPI and Docker.
5. **MLOps:** Experiment tracking, hyperparameter tuning, and pipeline automation.
